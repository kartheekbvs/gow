🔷 What is a Transformer?

A Transformer is a neural network architecture from the 2017 paper "Attention is All You Need" (Vaswani et al., Google Brain). It completely replaced RNNs and LSTMs and is now the backbone of every major AI system — GPT-4, BERT, Claude, Gemini.

The Core Insight

RNNs read text one word at a time, left to right. By the time they reach word 50, they've partly "forgotten" word 1. Transformers read all words at once and use attention to decide which words are relevant to each other — regardless of distance.

RNN vs Transformer — The fundamental difference

👁️ Attention — The Math

Attention is the single idea that makes Transformers work. This page breaks it down step by step with real numbers — exactly like the crio.do guide, but with more depth and interactive demos.

We'll use the sentence: "The bank near the river"

Why Do We Need Attention?

Consider "bank" — does it mean a financial institution or a riverbank? The word "river" is the clue. Attention learns to connect bank ↔ river automatically, even if they're far apart.

Attention(Q, K, V) = softmax( QKᵀ / √dk ) · V

The complete formula — Q finds matches, K offers matches, V provides content

Step 3 — Create Q, K, V Matrices

Each embedding is projected into three vectors using learned weight matrices W_Q, W_K, W_V. This is just matrix multiplication:

Q
Query — "What am I looking for?"
Each token asks: "Which other tokens are relevant to understanding me?" The query encodes the question.
Q = E · W_Q shape: [seq_len, d_k]
K
Key — "What do I have to offer?"
Each token announces: "Here's what information I can provide." The key describes each token's content type.
K = E · W_K shape: [seq_len, d_k]
V
Value — "Here's my actual content"
Once we know WHO to attend to (via Q·K), we grab WHAT from them (V). The value is the actual information transferred.
V = E · W_V shape: [seq_len, d_v]
pythonStep 3: Compute Q, K, V
import torch
import torch.nn as nn

d_model = 4   # embedding dimension (768 in real BERT)
d_k     = 4   # key/query dimension (64 in BERT)

# Learned weight matrices (randomly initialized, trained via backprop)
torch.manual_seed(42)
W_Q = torch.randn(d_model, d_k) * 0.1
W_K = torch.randn(d_model, d_k) * 0.1
W_V = torch.randn(d_model, d_k) * 0.1

# Project embeddings → Q, K, V
Q = embeddings @ W_Q   # [5,4] @ [4,4] = [5,4]
K = embeddings @ W_K   # [5,4] @ [4,4] = [5,4]
V = embeddings @ W_V   # [5,4] @ [4,4] = [5,4]

Step 4 — Attention Scores (Dot Product)

For each token, compute how similar its Query is to every other token's Key. High score = "pay attention to this token":

scores = Q · Kᵀ / √d_k
We divide by √d_k (square root of key dimension) to prevent gradients from vanishing when d_k is large
pythonStep 4: Attention Scores
import math

# Raw scores: every token attends to every other token
scores = torch.matmul(Q, K.transpose(-2, -1))  # [5, 5]
scores = scores / math.sqrt(d_k)              # scale

print("Scores shape:", scores.shape)  # [5, 5]
print("'bank' attends to each word:")
words = ["The", "bank", "near", "the", "river"]
for w, s in zip(words, scores[1].detach()):
    print(f"  → {w:8}: {s.item():.4f}")

Complete Attention Function

pythonFull attention implementation
import torch
import torch.nn.functional as F
import math

def scaled_dot_product_attention(Q, K, V, mask=None):
    """
    Args:
        Q: Query  [batch, heads, seq_len, d_k]
        K: Key    [batch, heads, seq_len, d_k]
        V: Value  [batch, heads, seq_len, d_v]
        mask: Optional mask [batch, 1, seq_len, seq_len]
    Returns:
        output:  [batch, heads, seq_len, d_v]  ← attended embeddings
        weights: [batch, heads, seq_len, seq_len]  ← attention matrix
    """
    d_k = Q.shape[-1]

    # 1. Compute similarity scores
    scores = torch.matmul(Q, K.transpose(-2, -1)) / math.sqrt(d_k)

    # 2. Apply mask (for decoder: hide future tokens)
    if mask is not None:
        scores = scores.masked_fill(mask == 0, -1e9)  # -∞ → softmax → 0

    # 3. Softmax → probability distribution
    weights = F.softmax(scores, dim=-1)

    # 4. Weighted sum of values
    output = torch.matmul(weights, V)

    return output, weights

# Test it

🎯 Multi-Head Attention

Running attention once gives one "perspective" on the relationships between words. Multi-Head Attention runs it H times in parallel, each head learning different relationships (subject-verb, coreference, syntax, etc.).

MultiHead(Q,K,V) = Concat(head₁, ..., headₕ) · Wₒ
where headᵢ = Attention(Q·Wᵢ_Q, K·Wᵢ_K, V·Wᵢ_V)
pythonMultiHeadAttention from scratch
import torch
import torch.nn as nn
import torch.nn.functional as F
import math

class MultiHeadAttention(nn.Module):
    def __init__(self, d_model=512, num_heads=8):
        super().__init__()
        assert d_model % num_heads == 0, "d_model must be divisible by num_heads"

        self.d_model    = d_model              # 512 in base Transformer
        self.num_heads  = num_heads            # 8 heads
        self.d_k        = d_model // num_heads # 64 dims per head

        # Each head gets its own Q, K, V projection
        self.W_Q = nn.Linear(d_model, d_model)  # covers all 8 heads at once
        self.W_K = nn.Linear(d_model, d_model)
        self.W_V = nn.Linear(d_model, d_model)
        self.W_O = nn.Linear(d_model, d_model)  # output projection

    def forward(self, x, mask=None):
        B, T, D = x.shape  # batch, seq_len, d_model
        H = self.num_heads

        # 1. Linear projections
        Q = self.W_Q(x)  # [B, T, D]
        K = self.W_K(x)
        V = self.W_V(x)

        # 2. Split into H heads: [B, T, D] → [B, H, T, d_k]
        Q = Q.view(B, T, H, self.d_k).transpose(1, 2)
        K = K.view(B, T, H, self.d_k).transpose(1, 2)
        V = V.view(B, T, H, self.d_k).transpose(1, 2)

        # 3. Scaled dot-product attention for ALL heads simultaneously
        scores  = torch.matmul(Q, K.transpose(-2, -1)) / math.sqrt(self.d_k)
        if mask is not None:
            scores = scores.masked_fill(mask == 0, -1e9)
        weights = F.softmax(scores, dim=-1)   # [B, H, T, T]
        out     = torch.matmul(weights, V)     # [B, H, T, d_k]

        # 4. Concatenate heads: [B, H, T, d_k] → [B, T, D]
        out = out.transpose(1, 2).contiguous().view(B, T, D)

        # 5. Final linear projection
        return self.W_O(out), weights

# Test it

📐 Embeddings

Embeddings are dense float vectors representing the meaning of text. Semantically similar texts → similar vectors → small cosine distance.

pythonsemantic_similarity.py
from transformers import AutoTokenizer, AutoModel
import torch, torch.nn.functional as F

model_name = "sentence-transformers/all-MiniLM-L6-v2"
tokenizer = AutoTokenizer.from_pretrained(model_name)
model     = AutoModel.from_pretrained(model_name)

def embed(texts):
    enc = tokenizer(texts, padding=True, truncation=True, return_tensors="pt")
    with torch.no_grad():
        out = model(**enc)
    # Mean pooling over token dimension
    emb = out.last_hidden_state.mean(dim=1)
    return F.normalize(emb, p=2, dim=1)  # L2 normalize
    # Returns: Tensor [batch, 384]

# Compute cosine similarity
e1 = embed(["I love playing with dogs"])
e2 = embed(["Puppies are my favourite animals"])
e3 = embed(["Stock market hit a new high"])

sim12 = (e1 * e2).sum().item()
sim13 = (e1 * e3).sum().item()
print(f"Dogs ↔ Puppies: {sim12:.3f}")   # high ~0.84
print(f"Dogs ↔ Stocks:  {sim13:.3f}")   # low  ~0.08

# Semantic search
queries = embed(["How to make pizza?"])
corpus  = embed([
    "Pizza dough recipe with yeast",
    "Best running shoes for marathons",
    "Italian cooking for beginners",
])
scores = (queries @ corpus.T)[0]
best   = scores.argmax().item()
print(f"Most relevant doc: {best}")  # 0 or 2

📍 Positional Encoding

Transformers process all tokens at once — they have no inherent sense of word order. Positional encoding injects position information by adding a unique vector to each embedding.

PE(pos, 2i) = sin(pos / 10000^(2i/d_model))
PE(pos, 2i+1) = cos(pos / 10000^(2i/d_model))
Even dimensions use sine, odd use cosine. Each position gets a unique pattern across all dimensions.
pythonpositional_encoding.py
import torch
import torch.nn as nn
import math

class PositionalEncoding(nn.Module):
    def __init__(self, d_model=512, max_seq_len=5000, dropout=0.1):
        super().__init__()
        self.dropout = nn.Dropout(dropout)

        # Build PE matrix once
        pe  = torch.zeros(max_seq_len, d_model)
        pos = torch.arange(0, max_seq_len).unsqueeze(1).float()
        div = torch.exp(torch.arange(0, d_model, 2).float()
                       * (-math.log(10000.0) / d_model))

        pe[:, 0::2] = torch.sin(pos * div)  # even dims
        pe[:, 1::2] = torch.cos(pos * div)  # odd dims
        pe = pe.unsqueeze(0)              # [1, max_seq_len, d_model]

        self.register_buffer("pe", pe)   # not a parameter, but saved with model

    def forward(self, x):
        # x: [batch, seq_len, d_model]
        x = x + self.pe[:, :x.size(1)]  # add position info
        return self.dropout(x)
        # Returns: [batch, seq_len, d_model] — same shape, now position-aware

pe_layer = PositionalEncoding(d_model=512)
x = torch.randn(2, 10, 512)
out = pe_layer(x)
print("Input shape: ", x.shape)    # [2, 10, 512]
print("Output shape:", out.shape)   # [2, 10, 512] — same
print("Changed?", not torch.allclose(x, out))  # True — positions added

Build a full Transformer encoder in pure PyTorch. This is the best way to truly understand the architecture — every layer, every weight, every forward pass.

pythontransformer_from_scratch.py — complete implementation
import torch
import torch.nn as nn
import torch.nn.functional as F
import math

# ─── 1. MULTI-HEAD ATTENTION ──────────────────────────
class MultiHeadAttention(nn.Module):
    def __init__(self, d_model, num_heads):
        super().__init__()
        self.d_k = d_model // num_heads
        self.h   = num_heads
        self.Wq  = nn.Linear(d_model, d_model)
        self.Wk  = nn.Linear(d_model, d_model)
        self.Wv  = nn.Linear(d_model, d_model)
        self.Wo  = nn.Linear(d_model, d_model)

    def forward(self, x, mask=None):
        B, T, D = x.shape
        Q, K, V = (self.Wq(x).view(B,T,self.h,self.d_k).transpose(1,2),
                   self.Wk(x).view(B,T,self.h,self.d_k).transpose(1,2),
                   self.Wv(x).view(B,T,self.h,self.d_k).transpose(1,2))
        s = torch.matmul(Q, K.transpose(-2,-1)) / math.sqrt(self.d_k)
        if mask is not None: s = s.masked_fill(mask==0, -1e9)
        out = torch.matmul(F.softmax(s,dim=-1), V)
        out = out.transpose(1,2).contiguous().view(B, T, D)
        return self.Wo(out)

# ─── 2. FEED-FORWARD NETWORK ──────────────────────────
class FeedForward(nn.Module):
    def __init__(self, d_model, d_ff=2048, dropout=0.1):
        super().__init__()
        self.net = nn.Sequential(
            nn.Linear(d_model, d_ff),
            nn.ReLU(),
            nn.Dropout(dropout),
            nn.Linear(d_ff, d_model)
        )
    def forward(self, x): return self.net(x)

# ─── 3. ENCODER LAYER ─────────────────────────────────
class EncoderLayer(nn.Module):
    def __init__(self, d_model, num_heads, d_ff, dropout=0.1):
        super().__init__()
        self.attn  = MultiHeadAttention(d_model, num_heads)
        self.ff    = FeedForward(d_model, d_ff, dropout)
        self.norm1 = nn.LayerNorm(d_model)
        self.norm2 = nn.LayerNorm(d_model)
        self.drop  = nn.Dropout(dropout)

    def forward(self, x, mask=None):
        # Residual connection: x + sublayer(norm(x))
        x = x + self.drop(self.attn(self.norm1(x), mask))  # pre-norm
        x = x + self.drop(self.ff  (self.norm2(x)))
        return x

# ─── 4. FULL ENCODER ──────────────────────────────────
class TransformerEncoder(nn.Module):
    def __init__(self, vocab_size, d_model=512, num_heads=8,
                       num_layers=6, d_ff=2048, max_len=512, dropout=0.1):
        super().__init__()
        self.embed = nn.Embedding(vocab_size, d_model)
        self.pe    = self._build_pe(max_len, d_model)
        self.layers = nn.ModuleList([
            EncoderLayer(d_model, num_heads, d_ff, dropout)
            for _ in range(num_layers)
        ])
        self.norm  = nn.LayerNorm(d_model)
        self.drop  = nn.Dropout(dropout)
        self.d_model = d_model

    def _build_pe(self, max_len, d_model):
        pe  = torch.zeros(max_len, d_model)
        pos = torch.arange(0, max_len).unsqueeze(1).float()
        div = torch.exp(torch.arange(0, d_model, 2).float() * (-math.log(10000) / d_model))
        pe[:, 0::2] = torch.sin(pos * div)
        pe[:, 1::2] = torch.cos(pos * div)
        return pe.unsqueeze(0)  # [1, max_len, d_model]

    def forward(self, x, mask=None):
        T = x.size(1)
        x = self.drop(self.embed(x) * math.sqrt(self.d_model)  # scale embeddings
                      + self.pe[:, :T].to(x.device))
        for layer in self.layers:
            x = layer(x, mask)
        return self.norm(x)
        # Returns: [batch, seq_len, d_model] — contextualized embeddings

# ─── 5. ADD CLASSIFICATION HEAD ───────────────────────
class TextClassifier(nn.Module):
    def __init__(self, vocab_size, num_classes, **encoder_kwargs):
        super().__init__()
        self.encoder = TransformerEncoder(vocab_size, **encoder_kwargs)
        self.head    = nn.Linear(encoder_kwargs.get("d_model", 512), num_classes)

    def forward(self, x, mask=None):
        enc  = self.encoder(x, mask)      # [B, T, d_model]
        cls  = enc[:, 0, :]             # take [CLS] token
        return self.head(cls)            # [B, num_classes]

from transformers import AutoTokenizer, AutoModelForCausalLM
import torch

class TransformerChatbot:
    def __init__(self, model_name="microsoft/DialoGPT-medium"):
        print("Loading chatbot model...")
        self.tokenizer = AutoTokenizer.from_pretrained(model_name)
        self.model     = AutoModelForCausalLM.from_pretrained(model_name)
        self.device    = "cuda" if torch.cuda.is_available() else "cpu"
        self.model     = self.model.to(self.device)
        self.model.eval()
        self.history   = None  # maintains conversation context
        print(f"Model ready on {self.device}")

    def respond(self, user_input: str, max_new_tokens=150) -> str:
        """Generate a response given user input + conversation history."""
        # Encode user input + EOS token
        new_ids = self.tokenizer.encode(
            user_input + self.tokenizer.eos_token, return_tensors="pt"
        ).to(self.device)

        # Append to history (so model has full conversation context)
        bot_input = (torch.cat([self.history, new_ids], dim=-1)
                     if self.history is not None else new_ids)

        # Limit context to avoid OOM (keep last 1000 tokens)
        if bot_input.shape[-1] > 1000:
            bot_input = bot_input[:, -1000:]

        # Generate response
        with torch.no_grad():
            output_ids = self.model.generate(
                bot_input,
                max_new_tokens=max_new_tokens,
                pad_token_id=self.tokenizer.eos_token_id,
                do_sample=True,       # enable randomness
                temperature=0.75,     # controls randomness
                top_p=0.9,            # nucleus sampling
                top_k=50,             # keep top 50 tokens
                repetition_penalty=1.1, # avoid repeating
                no_repeat_ngram_size=3
            )

        # Save full conversation as new history
        self.history = output_ids

        # Decode only the NEW tokens (the bot's response)
        new_tokens = output_ids[:, bot_input.shape[-1]:]
        response   = self.tokenizer.decode(new_tokens[0], skip_special_tokens=True)

        return response.strip() or "Sorry, I didn't understand that. Can you rephrase?"

    def reset(self):
        """Clear conversation history (start fresh)."""
        self.history = None

# Quick test
if __name__ == "__main__":

from flask import Flask, request, jsonify
from flask_cors import CORS
from chatbot import TransformerChatbot

app = Flask(__name__)
CORS(app)  # allow frontend to call backend

# Load model ONCE when server starts
bot = TransformerChatbot()

@app.route("/chat", methods=["POST"])
def chat():
    data    = request.get_json()
    message = data.get("message", "").strip()

    if not message:
        return jsonify({"error": "Message cannot be empty"}), 400

    if len(message) > 500:
        return jsonify({"error": "Message too long (max 500 chars)"}), 400

    response = bot.respond(message)
    return jsonify({"reply": response, "status": "ok"})

@app.route("/reset", methods=["POST"])
def reset():
    bot.reset()
    return jsonify({"status": "conversation cleared"})

@app.route("/health", methods=["GET"])
def health():
    return jsonify({"status": "ok", "model": "DialoGPT-medium"})

if __name__ == "__main__":
    app.run(debug=False, host="0.0.0.0", port=5000)
